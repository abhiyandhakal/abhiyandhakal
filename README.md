# Abhiyan Dhakal

I am a systems researcher based in Nepal. I am interested in making computing more efficient and predictable under resource constraints: using less memory and computation, remaining responsive under pressure, and degrading more gracefully when resources run out.

## Current systems research

My current work studies Linux memory pressure.

- **Broader goal:** How can memory-management policy preserve responsiveness when RAM is constrained?
- **Current milestone:** Can PSI identify early, repeatable pressure windows before reclaim and swap activity dominate?
- **Status:** Baseline and signal-validation experiments are in progress. Policy comparisons will follow after the signal is characterized across the workload matrix.

The work uses controlled workloads, repeated trials, stable raw experiment output, and explicit workload and memory controls.
